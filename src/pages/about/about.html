Diferecias:

Mostrar temperatura: El usuario tendrá acceso a ver la temperatura en el momento que el desee ya que la base de datos estará actualizada a cada cambio que se perciba en la estación meteorológica.
Cambio de datos entre mediciones: Se proveerá de un informe de cada cuanto tiempo cambia la temperatura para futuros análisis estadístico.
Control total del dispositivo: Encender o apagar su dispositivo desde cualquier parte que disponga de internet
